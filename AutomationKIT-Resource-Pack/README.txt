================================================================
  AUTOMATIONKIT — RESOURCE PACK
  Intro to Business Automation  ·  Josh O'Brien × Startup Vegas
================================================================

Everything you need to build the self-running client lifecycle
from the talk: a Notion business OS, the n8n automations, a full
setup guide, an AI prompt library, and a tooling guide.

----------------------------------------------------------------
WHAT'S INSIDE
----------------------------------------------------------------

1.  Step-by-Step-Setup-Guide.pdf
    START HERE. The full build, end to end — Notion, n8n,
    credentials, buttons, the landing-page form, and testing.

2.  Notion template (one-click clone) + Notion-Business-OS-Blueprint.pdf
    Clone the entire workspace into your Notion in one click:
      https://sustaining-verse-3bb.notion.site/Home-AutomationKIT-388c6115f2e08164bf62f9cd8b3cb4f9?source=copy_link
    Open it, then click Duplicate (top-right). The Blueprint PDF
    documents every property, option, and view if you'd rather
    build it by hand.

3.  n8n-workflows/
    Importable automation templates (Workflows > Import from File):
      01-lead-qualification.json   form -> Claude scores -> CRM
      02-client-onboarding.json    button -> client + project + email + tasks
      03-client-offboarding.json   button -> alumni + complete + wrap-up
    Replace the <<YOUR_..._DATABASE_ID>> placeholders and attach
    your own credentials (see the Setup Guide).

4.  AI-Prompt-Library.pdf
    40 production-ready prompts across Sales, Marketing,
    Operations, and Reporting — the same patterns that power
    the live demo.

5.  Tooling-Guide.pdf
    Comparable alternatives to Notion / n8n / Claude, plus the
    complementary tools that round out the stack, and how to choose.

----------------------------------------------------------------
SUGGESTED ORDER
----------------------------------------------------------------
  1) Read the Setup Guide.
  2) Clone the Notion template (link above) -> click Duplicate.
     (Or build it from the Blueprint PDF.)
  3) Import the three n8n workflows, add credentials, activate.
  4) Add the CRM buttons and wire your landing-page form.
  5) Test each flow, then make it yours.
